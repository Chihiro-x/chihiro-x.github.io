零散知识

try

用来处理异常，运行代码时有时会出现各种各样的错误，致使解析器中断执行，并提示xxxxxxErorr的提示，后面跟具体的错误的描述，这被称为是引发了异常。其实异常告诉我们的是遇到了什么类型的错误。xxxxErorr就是python内置的异常类型。每一种异常类型，都有它自己的错误信息。出现异常程序终止。其它有些异常类型不影响代码的功能，而又不想让异常中断执行时，就要对异常进行处理，而不让他中断程序：

例如：有一个字典里面存放光头强的信息，想取到它的资料，但具体都有什么资料我们不知道，如果输入的键不存在就会引发异常而中断执行。致使后面的代码也为无法执行。

information = {'name': '光头强', 'age': '20', 'gender': '男'}
print(information['name'])
print(information['sex'])
print(information['age'])
"""
运行结果
光头强
Traceback (most recent call last):
  File "/home/chihiro/Project/PyCharmProject/text.py", line 3, in <module>
    print(information['sex'])
KeyError: 'sex'
"""

那么就要捕获这些异常，绕过它，能让下面的代码继续运行。python提供了try语句来实现这些功能。

语法：

try:
    可能发生异常的代码
except 异常类型：
    捕获的异常后要执行的代码

示例：

information = {'name': '光头强', 'age': '20', 'gender': '男'}
try:
    print(information['name'])
    print(information['sex'])
except KeyError:
    print("sex不存在")
    print(information['age'])
"""
运行结果
光头强
sex不存在
20
"""

捕获所有异常：

引发异常的xxxxErorr都是python异常的类型。python中所有异常类型都是基类Exception的基类。那么在写代码的时候不用指定异常的具体类型，只要捕获基类就等于捕获了所有的异常，但输出的是具体的异常信息，这很有利于在大范围内找到具体的错误。也就是你不知道会产生什么样的错误类型，Exception是万能的。

示例：

information = {'name': '光头强', 'age': '20', 'gender': '男'}
try:
    print(information['name'])
    print(information['sex'])
except Exception as e:
    print(type(e))
    print("sex不存在")
    print(information['age']
"""
运行结果
光头强
<class 'KeyError'>
sex不存在
20
"""

异常里的else子句：

else子句是可选的，如果使用，必须放在所有的except子句之后。它是在try子句没有发生异常的情况执行。

information = {'name': '光头强', 'age': '20', 'gender': '男'}
try:
    print(information['name'])
except Exception as e:
    print(type(e))
else:
    print("光头强是伐木工")
"""
运行结果
光头强
光头强是伐木工
"""

finally子句

写在try语句的最后。不管 try 子句里面是否有异常，finally 子句都会执行。

information = {'name': '光头强', 'age': '20', 'gender': '男'}
try:
    print(information['name'])
    print(information['sex'])
except Exception as e:
    print(type(e))
    print("sex不存在")
    print(information['age'])
finally:
    print("调研结束")
"""
运行结果
光头强
<class 'KeyError'>
sex不存在
20
调研结束
"""

模块

为了编写可维护的代码，我们把很多函数分组，分别放到不同的文件里，这样，每个文件包含的代码就相对较少，很多编程语言都采用这种组织代码的方式。在Python中，一个.py文件就称之为一个模块（Module）。在Python中用关键字import来引入某个模块，比如要引用模块random，就可以在文件最开始的地方用import random来引入。在调用random模块中的函数时，必须这样引用：如random.randint()Python的from语句让你从模块中导入一个指定的部分到当前命名空间中，例如，要导入模块random的randint()函数，使用如下语句from ranfom import randint

random

random库是用于产生并运用随机数的标准库

randint()

生成指定区间内的随机整数。

示例

"""生成1～10之间的随机整数"""
import random
print (random.randint(1, 10))

seed()

主要是利用种子去产生一个随机数序列，指定的种子对应指定的随机数序列，如果不对种子赋值或者不调用seed函数，默认使用当前的时间作为种子去产生一个随机数序列

示例

"""
不设置种子
运行第一次结果0.7016987358817125
运行第二次结果0.7271754857060891
"""
import random
print(random.seed())
print(random.random())
"""
产生种子5对应的序列
第一次运行结果0.6229016948897019
第二次运行结果0.6229016948897019
"""
import random
print(random.seed(5))
print(random.random())

random()

生成0～1之间的随机小数

示例

import random
print(random.random())

requests()

用来做网页内容的解析

发送请求

GET请求

import requests
requests.get("https://www.sqlsec.com")

POST请求

import requests
data = {
    "username": "admin",
    "password": "password",
    "Login": "Login",
    "user_token": "3370d41cf95f4e9b08360f5fe9316325"
}
requests.post("http://192.168.31.54:222/login.php", data=data)

其他HTTP请求

import requests
requests.put('https://httpbin.org/put', data = {'key':'value'})

requests.delete('https://httpbin.org/delete')

requests.head('https://httpbin.org/get')

requests.options('https://httpbin.org/get')

参数传递

import requests
"直接将参数拼接在URL后面"
requests.get("http://192.168.31.54:222/Less-1?id=1")

"使用关键字params,等同于http://192.168.31.54:222/Less-1?id=1"
requests.get("http://192.168.31.54:222/Less-1", params = {"id":1})

"使用data,以表单的形式传递参数（参数不会拼接到 URL 后面）"
requests.post("http://192.168.31.54:222/Less-1", data = {"id":1})

"添加headers"
headers = {"User-Agent":"Mozilla/5.0"}
requests.get("http://192.168.31.54:222", headers=headers)

获取响应内容

import requests
re = requests.get("https://www.sqlsec.com")
"获取服务器响应文本内容"
print(re.text)
"获取服务器文本响应内容编码格式"
print(re.encoding)
"获取字节响应内容"
print(re.content)
"获取响应码"
print(re.status_code)
"获取响应头"
print(re.readers)

re

在 Python 中，我们可以使用内置的 re 模块来使用正则表达式。

re.match

语法：

re.match('匹配规则','需要过滤的内容')

示例：

"""提取结果123"""
import re
content = "hello 123 python"
re = re.match(r"^he.*\s(\d+)\s.*on$", content)
print(re.group(1))
"""
.*？是我们在匹配过程中最常使用到的
表示的就是匹配任意字符但是.*？的 . 代表所有的单个字符
除了 \n \如果我们的字符串有换行了怎么办呢？
这时候使用re.S,使 . 匹配包括换行在内的所有字符
"""
import re
content = r"hello \
123 python"
re = re.match(r"^he.*(\d{3}).*on$", content, re.S)
print(re.group(1))

注：re.group(N) 返回第N组括号匹配的字符，而re.group() == re.group(0) == 所有匹配的字符，re.groups()返回所有括号匹配的字符。

re.search

直接去扫描字符串，然后把匹配成功的地一个结果返回给你。

语法：

re.search('匹配规则','待扫描的内容')

示例：

"提取结果123"
import re
content = r"hello \
123 python"
re = re.search(r"^he.*(\d{3}).*on$", content, re.S)
print(re.group(1))

re.findall

通过它我们就能轻松的获取所有匹配的内容了

语法：

re.findall('匹配规则','待查找的内容')

示例：

"提取结果['123', '456', '789']"
import re
content = """hello 123 python;
hello 456 python;
hello 789 python;"""
res = re.findall(r'he.*?(\d+)\s.*?n;', content, re.S)
print(res)

re.sub

把匹配到的内容替换为其他内容

语法：

re.sub('匹配规则','替换为','待查找的内容')

示例：

"提取结果999 999 999"
import re
content = """hello 123 python;
hello 456 python;
hello 789 python;"""
res = re.sub(r'he.*?(\d+)\s.*?n;', '999', content, re.S)
print(res)

re.compile

这个主要就是把我们的匹配符封装一下

示例：

import re
content = "hello 123 python"
pattern = re.compile(r"^he.*\s(\d+)\s.*on$", re.S)
re = re.match(pattern, content)
print(re.group(1))

BeautifulSoup

可以从 HTML 或 XML 文件中提取数据的 Python 库

首先写一段HTML代码用来练习

html = """
<html>
<head>
    <title>学习爬虫</title>
    <meta charset="UTF-8">
</head>
<p>你是猪吗？</p>
<p>
    <a href="www.baidu.com">猪猪侠</a>
    <a href="www.baidu.com">狗熊岭</a>
    你喜欢哪一个？
</p>
</html>
"""

示例：

from bs4 import BeautifulSoup
html = """
<html>
<head>
    <title>学习爬虫</title>
    <meta charset="UTF-8">
</head>
<p>你是猪吗？</p>
<p>
    <a href="www.chihiro.com" id="demo">猪猪侠</a>
    <a href="https://chihiro-x.github.com">狗熊岭</a>
    你喜欢哪一个？
</p>
</html>
"""
soup = BeautifulSoup(html, 'lxml')
"获取标题内容"
print(soup.title.string)
"获取p标签里的内容"
print(soup.p.string)
"获取title的父级标签"
print(soup.title.parent.name)
"获取超链接，获取a标签内容使用soup.a.string"
print(soup.a)
"获取所有超链接"
print(soup.find_all('a'))
"获取id为demo的超链接"
print(soup.find(id="demo"))
"获取网页中的所有内容"
print(soup.get_text)

函数

range()

可创建一个整数列表

函数语法

range(start, stop step)

参数说明：

  • start: 计数从 start 开始。默认是从 0 开始。例如range（5）等价于range（0， 5）;
  • stop: 计数到 stop 结束，但不包括 stop。例如：range（0， 5） 是[0, 1, 2, 3, 4]没有5
  • step：步长，默认为1。例如：range（0， 5） 等价于 range(0, 5, 1)

示例

"输出1 2 3 4 5 6 7 8 9"
for x in range(1,10):
    print(x)
